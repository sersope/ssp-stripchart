A GTK+ widget for graphical representation of values over time.
 
Requeriments: PyGObject (see https://wiki.gnome.org/Projects/PyGObject).
For python 2 and 3.
Run this in a python console for an example:
    import ssp_stripchart
    ssp_stripchart.test()
